# CSLA Iron Curtain Compatibility

## Overview

This compatibility module adds support for CSLA Iron Curtain Czechoslovakian civilian units to the Civilian Presence system.

## Purpose

Provides authentic Cold War-era Czechoslovakian civilian population for missions set during the Iron Curtain period in Central Europe.

## Civilian Presets

### Czechoslovakian Civilians (CIV_CSLA)

- **Units**: 23 active civilian units
- **Period**: Cold War era (1980s)
- **DLC**: CSLA Iron Curtain
- **Includes**:
  - Citizens (4 variants)
  - Doctors
  - Foremen (2 variants)
  - Functionaries (2 variants)
  - Police personnel (2 active variants)
  - Villagers (4 variants)
  - Woodlanders (4 variants)
  - Workers (4 variants)

## Total Units

**23 active agent classes** (24 total, 1 disabled) in 1 Czechoslovakian civilian preset

## Disabled Units

- `CSLA_CIV_PoliceManCrw` - Police crew member

**Reason**: This specialized police crew variant is disabled to focus on general police presence (standard policeman and armed Sa61 variant).

## Notes

- All units require the CSLA Iron Curtain DLC
- Police units include crew variants and armed police with Sa61 submachine guns
