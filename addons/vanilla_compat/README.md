# Vanilla Arma 3 Compatibility

## Overview

This compatibility module adds support for vanilla Arma 3 IDAP (International Development & Aid Project) civilian units to the Civilian Presence system.

## Purpose

Enables the use of IDAP humanitarian workers and specialists in ambient civilian modules for operations involving aid organizations.

## Civilian Presets

### IDAP Personnel (CIV_IDAP)

- **Units**: 17 civilian units
- **DLC**: Arma 3 Laws of War
- **Includes**:
  - Aid workers (9 variants)
  - EOD specialists (2 units)
  - Paramedics
  - Pilots
  - UAV operators (4 variants including antimine and medical specialists)

## Total Units

**17 agent classes** in 1 IDAP civilian preset

## Notes

- All units require the Arma 3 Laws of War DLC (Orange DLC)
- Includes specialized personnel like EOD technicians and medical UAV operators
