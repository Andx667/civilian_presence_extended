# Community Factions Project (CFP) Compatibility

## Overview

This compatibility module adds support for Community Factions Project (CFP) civilian units to the Civilian Presence system.

## Purpose

Provides diverse civilian populations from various cultural and geographical regions for enhanced mission immersion.

## Civilian Presets

### Afghanistan Civilians (CIV_CFP_AFG)

- **Units**: 2 civilian units
- **Region**: Afghanistan

### African Christian Civilians (CIV_CFP_AFRCHRISTIAN)

- **Units**: 14 civilian units
- **Region**: Africa (Christian communities)

### African Islamic Civilians (CIV_CFP_AFRISLAMIC)

- **Units**: 12 civilian units
- **Region**: Africa (Islamic communities)

### Asian Civilians (CIV_CFP_ASIA)

- **Units**: 12 civilian units
- **Region**: Asia

### Chernarus Winter Civilians (CIV_CFP_CHERNO_WIN)

- **Units**: 3 civilian units
- **Region**: Chernarus (winter variants)

### Middle East Civilians (CIV_CFP_ME)

- **Units**: 2 civilian units
- **Region**: Middle East

## Total Units

**45 agent classes** across 6 regional civilian presets
