# S.O.G. Prairie Fire Compatibility

## Overview

This compatibility module adds support for S.O.G. Prairie Fire Vietnamese civilian units to the Civilian Presence system.

## Purpose

Provides authentic Vietnam War-era Vietnamese civilian population for period-accurate missions set in Southeast Asia.

## Civilian Presets

### Vietnamese Civilians (CIV_SOG_VN)

- **Units**: 32 civilian units
- **Period**: Vietnam War era (1960s-1970s)
- **Includes**: Various Vietnamese civilian models representing the local population
- **Region**: Vietnam

## Total Units

**32 agent classes** in 1 Vietnamese civilian preset

## Notes

- All units  equire the S.O.G Prarie Fire CDLC
