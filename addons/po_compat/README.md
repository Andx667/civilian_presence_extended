# Project OPFOR Compatibility

## Overview

This compatibility module adds support for Project OPFOR civilian units to the Civilian Presence system.

## Purpose

Provides diverse civilian populations from Africa, Middle East, and Eastern Europe for enhanced mission realism.

## Civilian Presets

### African Civilians (CIV_LOF_AFR)

- **Units**: 12 civilian units
- **Includes**: 6 base civilian models with both regular and slim (_S) variants
- **Region**: Africa

### Middle Eastern Civilians (CIV_LOF_TAK)

- **Units**: 16 civilian units
- **Includes**: Various Takistani civilian models plus a random variant
- **Region**: Takistan/Middle East

### Eastern European Civilians (CIV_LOF_CHR)

- **Units**: 32 civilian units
- **Includes**: Assistants, citizens, doctors, functionaries, policemen, priests, profiteers, rockers, school teachers, villagers, woodlanders, and workers
- **Region**: Chernarus/Eastern Europe

## Total Units

**60 agent classes** across 3 regional civilian presets
