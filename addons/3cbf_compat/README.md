# 3CB Factions Compatibility

## Overview
This compatibility module adds support for 3CB Factions civilian units to the Civilian Presence system.

## Purpose
Enables mission makers to use 3CB Factions civilians in ambient civilian modules for enhanced realism in various geographical settings.

## Civilian Presets

### African Desert Civilians (CIV_3CB_Factions_ADC)
- **Units**: 21 civilian units
- **Includes**: Christian and Islamic variants, hunters, laborers, doctors, functionaries, pilots, workers, villagers, and woodlanders

### Middle Eastern Civilians (CIV_3CB_Factions_MEC)
- **Units**: 14 civilian units
- **Includes**: Hunters, citizens, coaches, professionals, functionaries, helicopter pilots, laborers, doctors, civilian pilots, spotters, and workers

### Chernarus Civilians (CIV_3CB_Factions_CHC)
- **Units**: 18 civilian units
- **Includes**: Activists, bodyguards, citizens, coaches, doctors, functionaries, hikers, laborers, pilots, politicians, priests, professionals, spies, villagers, woodlanders, and workers

### Takistan Civilians (CIV_3CB_Factions_TKC)
- **Units**: 5 civilian units
- **Includes**: General civilians, doctors, pilots, spotters, and workers

## Total Units
**64 agent classes** across 4 regional civilian presets

## Commented Out Units
Currently, all units are active. No units are commented out.
